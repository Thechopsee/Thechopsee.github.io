Zásady ochrany soukromí

Úvod

Tato aplikace respektuje soukromí uživatelů a je navržena tak, aby nevyžadovala ani nezpracovávala osobní údaje.

Shromažďování a používání údajů

Aplikace neshromažďuje žádné osobní údaje uživatelů, jako jsou jména, e-mailové adresy nebo jiné identifikátory.

Veškerá data používaná aplikací se týkají výhradně výsledků lodních soutěží. Tato data nejsou nijak spojena s konkrétní fyzickou osobou, ale pouze s loděmi.

Uživatelské účty

Aplikace nevyžaduje registraci ani vytvoření uživatelského účtu.

Synchronizace dat

Aplikace může synchronizovat data s externími uzly (servery), které si uživatelé provozují sami nebo ve skupině. Tato komunikace slouží pouze k přenosu technických a soutěžních dat a neobsahuje osobní údaje.

Sdílení dat

Aplikace nesdílí žádná data třetím stranám.

Zabezpečení

I přesto, že aplikace nezpracovává osobní údaje, je dbáno na bezpečný přenos dat mezi zařízeními a servery.

Změny zásad

Tyto zásady mohou být v budoucnu aktualizovány. Aktuální verze bude vždy dostupná na této stránce.

Privacy Policy

Introduction

This application respects user privacy and is designed not to collect or process any personal data.

Data Collection and Use

The application does not collect any personal data such as names, email addresses, or other personal identifiers.

All data used within the application relates exclusively to sailing competition results. These data are not linked to any individual person, but only to boats.

User Accounts

The application does not require user registration or the creation of user accounts.

Data Synchronization

The application may synchronize data with external nodes (servers), which are operated by users individually or within a group. This communication is used solely for transferring technical and competition-related data and does not include personal data.

Data Sharing

The application does not share any data with third parties.

Security

Even though the application does not process personal data, reasonable measures are taken to ensure secure data transmission between devices and servers.

Changes to This Policy

This Privacy Policy may be updated in the future. The current version will always be available on this page.
